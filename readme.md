This project, "Drum ü•Å Kit," is a simple web application that allows users to produce seven different drum sounds using keyboard keys or by clicking on the corresponding drum images.

## Project Overview

### Technologies Used
- HTML
- CSS
- JavaScript

### Features
- Seven drum sounds
- Play sounds using keyboard keys or mouse clicks
- Visual representation with drum images

## Getting Started

### Prerequisites
- Web browser (Chrome, Firefox, Safari, etc.)

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/your-username/drum-kit.git
    ```
2. Open the `index.html` file in your preferred web browser.

### Usage
- Press the following keys to produce drum sounds:
    - `W` - Snare
    - `A` - Kick
    - `S` - Crash
    - `D` - Hi-Hat
    - `J` - Tom
    - `K` - Ride
    - `L` - Floor Tom
- Click on the drum images to produce sounds.
